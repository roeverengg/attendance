# College Attendance Management System

A Google Apps Script based attendance system for colleges.

## Features
- Class advisor login
- Date & hour-wise attendance
- Automatic 7-hour daily columns
- Present by default
- Google Sheet backend

## Tech Stack
- Google Apps Script
- Google Sheets
- HTML, CSS, JavaScript
